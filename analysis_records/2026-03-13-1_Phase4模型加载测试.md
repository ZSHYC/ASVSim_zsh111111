## 遇到的问题与解决

### 问题 1: 缺少 safetensors 库

**症状**:
```
ModuleNotFoundError: No module named 'safetensors'
```

**解决**:
```bash
pip install safetensors
```

### 问题 2: transformers 无法识别 DA3

**症状**:
```
Unrecognized model in D:\ASVSim_models\depth_anything_3
Should have a `model_type` key in its config.json
```

**原因**: DA3 模型只有权重文件，没有配置文件

**解决**: 使用 safetensors.torch.load_file() 直接加载权重
