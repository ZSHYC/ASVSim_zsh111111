## 4. DA3 官方 API 发现

**关键发现** (来自 README):

### 模型加载
```python
from depth_anything_3.api import DepthAnything3
import torch

device = torch.device("cuda")
model = DepthAnything3.from_pretrained("depth-anything/DA3NESTED-GIANT-LARGE")
model = model.to(device=device)
```

### 推理输出
```python
prediction = model.inference(images)

# 输出包含:
prediction.depth          # [N, H, W] depth map
prediction.conf           # [N, H, W] confidence
prediction.extrinsics     # [N, 3, 4] camera pose (opencv format)
prediction.intrinsics     # [N, 3, 3] camera intrinsics
prediction.processed_images  # [N, H, W, 3] processed input
```

### 命令行使用
```bash
# 使用 backend 模式（缓存模型到 GPU）
da3 backend --model-dir depth-anything/DA3NESTED-GIANT-LARGE --gallery-dir workspace/gallery

# 自动处理文件夹
da3 auto path/to/images --export-format glb --export-dir output/

# 视频处理
da3 video input.mp4 --fps 15 --export-format glb-feat_vis
```
