## 根本原因分析

### 问题1: 光晕/暗角 (Vignette)

**错误配置** (已移除):
```json
"PostProcessSettings": {
  "bOverride_VignetteIntensity": true, "VignetteIntensity": 0.0,
  "bOverride_ChromaticAberrationIntensity": true, "ChromaticAberrationIntensity": 0.0,
  ...
}
```

**问题**: `PostProcessSettings` **不是ASVSim/AirSim支持的有效配置项**！

根据ASVSim官方文档，CameraDefaults.CaptureSettings **只支持**:
| 参数 | 类型 | 说明 |
|------|------|------|
| ImageType | int | 图像类型 (0=Scene, 1=DepthPlanar, 5=Segmentation) |
| Width | int | 图像宽度 |
| Height | int | 图像高度 |
| FOV_Degrees | float | 视场角 |
| MotionBlurAmount | float | 运动模糊 (0=禁用) |
| ChromaticAberrationScale | float | 色差强度 (0=禁用) |
| AutoExposureBias | float | 自动曝光偏差 |
| AutoExposureMaxBrightness | float | 自动曝光最大亮度 |
| AutoExposureMinBrightness | float | 自动曝光最小亮度 |
| LumenGIEnable | bool | Lumen全局光照 |
| LumenReflectionEnable | bool | Lumen反射 |

**重要**: `Vignette`, `Bloom`, `LensFlare` **无法通过settings.json配置禁用**！

这些是UE5 SceneCaptureComponent的默认后期处理效果，ASVSim没有提供配置接口。

### 问题2: 船体仍然可见

**原配置**:
```json
"front_camera": {
  "X": 2.5, "Z": -1.8, "Pitch": -10.0, "FOV_Degrees": 90
}
```

**问题分析**:
1. **FOV 90°太宽** - 超广角能看到两侧船舷
2. **Z:-1.8不够高** - 仍能看到甲板
3. **Pitch -10°下倾** - 视线向下更容易看到船体
4. **X:2.5不够前** - 相机还在船头附近

## 修复方案

### 修复1: 移除无效PostProcessSettings配置

**正确配置** (仅使用支持的参数):
```json
"CameraDefaults": {
  "CaptureSettings": [
    {
      "ImageType": 0, "Width": 640, "Height": 480, "FOV_Degrees": 70,
      "LumenGIEnable": false, "LumenReflectionEnable": false,
      "MotionBlurAmount": 0,
      "ChromaticAberrationScale": 0.0,
      "AutoExposureBias": 0.0
    },
    ...
  ]
}
```

**变更**:
- ✅ 移除无效的 `PostProcessSettings` 块
- ✅ 添加 `ChromaticAberrationScale: 0.0` (正确的参数名)
- ✅ 添加 `AutoExposureBias: 0.0`

### 修复2: 调整相机位置去除船体

**新配置**:
```json
"front_camera": {
  "X": 3.5,      // 前移1米，越过船头
  "Z": -2.5,     // 抬高0.7米，越过桅杆
  "Pitch": 0.0,  // 水平视角，不下倾
  "FOV_Degrees": 70  // 减小视场角，视野更窄
}
```

**变更**:
- X: 2.5 → **3.5** (前移1米)
- Z: -1.8 → **-2.5** (抬高0.7米)
- Pitch: -10.0 → **0.0** (水平视角)
- FOV: 90 → **70** (减小视野)

## 关于光晕的重要说明

### ⚠️ 无法完全禁用

**Vignette (暗角)** 和 **Bloom (泛光)** 是UE5 SceneCaptureComponent的默认后期处理效果。

ASVSim **不提供配置接口**来禁用这些效果。

### 缓解方案

| 方案 | 可行性 | 说明 |
|------|--------|------|
| 代码层禁用 | ❌ 不可行 | 需要修改ASVSim C++源码 |
| 配置禁用 | ❌ 不可行 | 不支持相关配置参数 |
| 图像后处理 | ✅ 可行 | 采集后用OpenCV去除暗角 |
| 忽略第一帧 | ✅ 可行 | 第一帧光晕最重，从第二帧开始用 |

**推荐方案**: 忽略第一帧（光晕最重），或采集后用OpenCV去除暗角。

## 如果船体仍然可见

如果调整后的相机位置仍然能看到船体，需要进一步前移/抬高：

```json
"front_camera": {
  "X": 4.5,      // 再前移1米
  "Z": -3.0,     // 再抬高0.5米
  "Pitch": 5.0,  // 轻微上扬，看更远
  "FOV_Degrees": 60  // 进一步减小视野
}
```

**注意**: 相机位置过高会导致只能看到天空，需要平衡。
