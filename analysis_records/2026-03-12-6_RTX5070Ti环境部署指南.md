## 快速部署步骤

### Step 1: 创建 D 盘模型目录

```powershell
# 以管理员身份运行 PowerShell
New-Item -ItemType Directory -Force -Path "D:\ASVSim_models\sam3"
New-Item -ItemType Directory -Force -Path "D:\ASVSim_models\depth_anything_3"
New-Item -ItemType Directory -Force -Path "D:\ASVSim_models\cache\hub"
New-Item -ItemType Directory -Force -Path "D:\ASVSim_models\datasets"
```

### Step 2: 创建 bishe Conda 环境 (C盘)

```bash
# 打开 Anaconda Prompt 或 Git Bash
conda create -n bishe python=3.12 -y
conda activate bishe

# 升级 pip
python -m pip install --upgrade pip setuptools wheel
```

### Step 3: 安装 PyTorch 2.6 (C盘环境目录)

```bash
# PyTorch 会安装在 conda 环境目录（通常在 C 盘用户目录下）
# 这是正常的，因为 PyTorch 是运行库
pip install torch==2.6.0 torchvision==0.21.0 torchaudio==2.6.0 --index-url https://download.pytorch.org/whl/cu126
```

### Step 4: 安装其他依赖 (C盘)

```bash
pip install numpy scipy scikit-learn scikit-image matplotlib tqdm
pip install opencv-python Pillow imageio pyyaml tensorboard
pip install huggingface_hubhf-transfer
```

### Step 5: 配置 Hugging Face 缓存到 D 盘

```bash
# 设置环境变量，让 Hugging Face 下载到 D 盘
conda env config vars set HF_HOME="D:\ASVSim_models\cache\hub"
conda env config vars set TORCH_HOME="D:\ASVSim_models\cache\torch"
conda env config vars set PYTHONPATH="C:\Users\zsh\Desktop\ASVSim_zsh"

# 重新激活环境使变量生效
conda deactivate
conda activate bishe
```

### Step 6: 验证安装

```bash
python -c "
import torch
import os

print('='*60)
print('PyTorch 版本:', torch.__version__)
print('CUDA 版本:', torch.version.cuda)
print('GPU:', torch.cuda.get_device_name(0))
print('显存:', f'{torch.cuda.get_device_properties(0).total_memory / 1e9:.2f} GB')
print()
print('HF_HOME:', os.environ.get('HF_HOME', 'Not set'))
print('TORCH_HOME:', os.environ.get('TORCH_HOME', 'Not set'))
print()

# 简单测试
x = torch.rand(1000, 1000).cuda()
y = torch.matmul(x, x)
print('CUDA 测试: ✅ 通过')
print('='*60)
"
```

## 模型下载指南

### SAM 3 权重下载

**方法1: 通过 Hugging Face (推荐)**
```bash
# 先申请权限: https://huggingface.co/facebook/sam3
# 登录 Hugging Face
huggingface-cli login

# 下载到 D 盘
python -c "
from huggingface_hub import hf_hub_download
model_path = hf_hub_download(
    repo_id='facebook/sam3',
    filename='sam3.pt',
    local_dir='D:\\ASVSim_models\\sam3',
    local_dir_use_symlinks=False
)
print(f'Downloaded to: {model_path}')
"
```

**方法2: 手动下载**
1. 访问 https://huggingface.co/facebook/sam3
2. 申请权限并下载 `sam3.pt`
3. 放到 `D:\ASVSim_models\sam3\sam3.pt`

### Depth Anything 3 权重下载

**方法1: 通过 Hugging Face (推荐)**

**方法2: 手动下载**

## 使用说明

### 1. 一键部署 (推荐)

```powershell
# 以管理员身份运行 PowerShell
# 进入到项目目录
cd C:\Users\zsh\Desktop\ASVSim_zsh

# 运行部署脚本
.\config\setup_bishe_env.bat
```

### 2. 手动部署

```bash

conda create -n bishe python=3.12 -y
conda activate bishe

# 安装依赖

```

