## 4. DA3 官方 API 测试

### 第一次尝试 (GPU)

**问题**: CUDA 错误
```

```

**原因**: PyTorch 2.6 不完全支持 RTX 5070 Ti (sm_120)

## 6. 遇到的问题与解决

### 问题 1: 缺少 addict 模块

**症状**:
```
ModuleNotFoundError: No module named 'addict'
```

**解决**:
```bash
pip install addict
```

### 问题 2: CUDA 兼容性错误

**症状**:
```
RuntimeError: CUDA error: no kernel image is available for execution on the device
```

**原因**: PyTorch 2.6 不完全支持 RTX 5070 Ti (sm_120)

**解决方案**:
1. **临时方案**: 使用 CPU 推理 (当前使用)
2. **长期方案**: 升级到 PyTorch Nightly (CUDA 12.8)

### 问题 3: 缺少 triton (警告)

**症状**:
```
A matching Triton is not available, some optimizations will not be enabled
```

**影响**: 轻微，只是优化未启用

**解决**: 可选，安装 triton 可获得更好性能

## 7. 关键发现

### DA3 输出包含

1. **depth**: [N, H, W] 深度图 (米)
2. **conf**: [N, H, W] 置信度
3. **extrinsics**: [N, 3, 4] 相机位姿 (OpenCV format)
4. **intrinsics**: [N, 3, 3] 相机内参
5. **processed_images**: [N, H, W, 3] 预处理后的图像

### 模型参数

- **总参数量**: 410.9M (约 411M)
- **模型大小**: ~1.5 GB
- **架构**: DINOv2 ViT + Depth Head

### CPU 性能

- ** surprisingly fast**: 1.72s 单张图像
- **预处理**: 0.02s
- **前向传播**: 1.70s
- **后处理**: 0.00s
