## 7. 与深度/位姿融合建议

### 融合策略

1. **实例-深度关联**
   ```python
   # 对每个实例，计算平均深度
   instance_depths = []
   for instance_id in range(1, num_instances+1):
       mask = (instance_mask == instance_id)
       depth_values = da3_depth[mask]
       mean_depth = depth_values.mean()
       instance_depths.append(mean_depth)
   ```

2. **近处实例过滤**
   - 船体实例：深度 < 2m
   - 浮冰实例：深度 2-20m
   - 远处背景：深度 > 20m

3. **时序一致性**
   - 跟踪实例在连续帧中的ID
   - 关联同一物体的时间序列
