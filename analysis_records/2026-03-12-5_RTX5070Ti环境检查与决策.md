## 部署方案决策

### 推荐方案: PyTorch 2.6 Stable + CUDA 12.6/13.1

**理由**:
1. 驱动版本 591.74 支持 CUDA 13.1
2. PyTorch 2.6+ 已支持 CUDA 12.6，兼容性好
3. 无需使用 Nightly 版本，稳定性更高
4. 12GB 显存可运行 DA3-LARGE (0.35B) 和 SAM 3

**PyTorch 版本选择**:
- PyTorch 2.6.0 (Stable) - 推荐
- PyTorch 2.7.0 (Preview) - 如 2.6 有问题

### 安装命令

```bash
# 创建 bishe 环境
conda create -n bishe python=3.12 -y
conda activate bishe

# 安装 PyTorch 2.6 + CUDA 12.6 (稳定版)
pip install torch==2.6.0 torchvision==0.21.0 torchaudio==2.6.0 \
    --index-url https://download.pytorch.org/whl/cu126

# 或尝试 CUDA 13.1 (如果可用)
# pip install torch torchvision torchaudio \
#     --index-url https://download.pytorch.org/whl/cu131
```

## 模型显存预算 (12GB)

### 同时加载多个模型的显存分配

| 模型 | 参数量 | 推理显存 | 备注 |
|------|--------|----------|------|
| **DA3-LARGE** | 0.35B | ~4GB | 首选深度估计 |
| **SAM 3** | 0.85B | ~3GB | 实例分割 |
| PyTorch/CUDA 开销 | - | ~2GB | 基础开销 |
| **总计** | - | **~9GB** | 在 12GB 范围内 ✅ |

**建议**:
- 不同时加载 DA3 和 SAM 3（除非必要）
- 使用 `torch.cuda.empty_cache()` 在切换模型时清理显存
- 单帧推理，无需 batch

   ```bash
   python -c "import torch; print(f'PyTorch: {torch.__version__}'); \

              print(f'GPU: {torch.cuda.get_device_name(0)}'); \
   ```
