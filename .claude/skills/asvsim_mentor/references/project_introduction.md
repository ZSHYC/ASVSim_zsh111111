# Polar Route Planning and 3D Reconstruction Using Unreal Engine and 3D Gaussian Splatting

## 项目背景

随着全球气候变化和极地航道开发需求的不断增长，极地航行及冰区作业活动日益频繁，极地船舶航行安全保障已成为国际海事领域的重大挑战。极地环境具有低照度、天气多变等恶劣特点，现有的船舶路径规划方法在面对复杂多变的极地冰水环境时，往往缺乏对冰情动态变化的实时响应能力，无法有效平衡航行安全性与航行效率的多目标优化需求。
为解决极地数据难于获取及船舶航行的路径规划与实时避碰等技术难题，本项目利用Unreal引擎渲染高质量极地环境数据，提出基于视觉与雷达多传感器融合的半监督深度估计与海冰实例分割模型，基于全局路径规划与局部避碰算法实时规划船舶运动并与Unreal引擎动态交互。
本项目创新性地提出基于三维高斯散点渲染（3D Gaussian Splatting）技术的冰水环境实时重构方法，通过隐式表达建模突破了传统基于纹理匹配的场景重构局限。该技术能够为纹理稀疏、形态复杂的极地目标提供高精度三维建模，为基于精确环境感知的船舶智能路径规划提供了全新的技术支撑，对提升极地航行安全性和自主导航能力具有重要的研究意义和应用价值。

## 研究内容

**unreal生成数据→智能感知→3DGS重建→路径规划**
本项目围绕极地冰水环境下的船舶安全航行需求，构建了一个完整的从环境感知到路径规划的技术链条。首先在Unreal引擎下，构建极地冰水环境，开展基于SAM3的海冰实例分割模型研究，开展相机与雷达的空间联合标定，并研究基于半监督学习的海冰深度估计模型（基于Depth Anything 3），推断海冰的3D空间位置，构建多信息融合的智能感知系统，为后续路径规划提供准确的目标位置和属性信息。
在环境重构层面，项目开发基于多尺度渐进式三维高斯泼溅（3DGS）的冰水环境重构技术和基于分块建模与深度融合的大场景重建技术。
多尺度渐进式3DGS模型采用层次化建模策略，通过浅层残差块拟合远距粗视图以捕捉大尺度背景结构，深层残差块专注于学习近距高频特征以刻画目标表面细节，实现对冰山、冰面等复杂目标的高精度三维重构。
针对广阔的极地航行场景，采用分块建模策略将大场景划分为多个相互重叠的子区域，对每个区域独立进行3DGS建模以提升处理效率。利用深度相机、激光雷达等设备提供的深度信息，通过深度融合视图重建优化算法将各子模型精确拼接，并结合循环神经网络RNN进行无监督优化，修复模型缺陷，构建完整连贯的大范围环境模型。
基于前述环境感知和重构技术，项目的核心在于开发极地冰水环境下的船舶智能路径规划技术。该技术综合考虑海冰密集度、厚度等冰情参数，构建动态路径调整机制，当实时目标检测发现新的冰情变化时，系统能够触发路径重规划，支持复杂冰情条件下的实时航行决策，为极地船舶提供安全、高效的智能导航支持。

## Expected Outcome

预期完成一篇博士毕业论文，系统阐述基于Unreal引擎的极地船舶路径规划算法与3DGS技术的环境重构方法的设计与实现。
